# 🌡️ IoT Temperature Monitoring System (Python)

## 📌 Description
Real-time IoT Temperature Monitoring System with Live Graph using Python

## ⚙️ Features
- Temperature data simulation
- Threshold-based alert system
- Graph visualization
- Continuous monitoring

## 🧠 Concept
This project is based on:
- ECE: Sensor data & signal processing
- IoT: Real-time monitoring system

## ▶️ How to Run
1. Install required libraries:
   pip install matplotlib

2. Run:
   python Virtual Temperature live Monitoring System.py

## 📊 Output
- Temperature readings will be live in.    console
- Graph visualization
